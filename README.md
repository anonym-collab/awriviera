# AW Riviera

Site vitrine one-page — **Yacht Brokerage & Luxury Charter** en Méditerranée.
Refonte luxe & épurée. Page unique, sans dépendance (HTML/CSS/JS natif).

## Aperçu
Ouvrez simplement `index.html` dans un navigateur.

## Sections
Hero · Approche (ADN + 4 piliers) · Services (3) · Destinations (8) ·
Process (4 étapes) · Traiteur (Déjeuner / Boissons / Dîner) · Contact · Footer.

## Fonctionnalités
- Bilingue **FR / EN** (sélecteur en haut à droite, langue mémorisée).
- Animations d'apparition au défilement.
- Remplacement des photos de destinations / traiteur depuis le navigateur.
- Ajout de menus PDF (Déjeuner / Boissons / Dîner), stockés localement.

## Personnalisation
- **Couleurs** : variables CSS `:root` en haut du `<style>` (`--navy`, `--gold`, …).
- **Photos** : survolez une carte et cliquez l'icône en haut à droite pour
  remplacer l'image (ou modifiez les URL `--dest-img` / `background-image`).
- **Coordonnées** : section `#contact` (email, WhatsApp, base).
